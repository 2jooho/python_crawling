# 파이썬을 통해 크롬을 실행시켜 키워드 검색 후 10여 페이지 이동하며 크롤링

chromedriver.exe를 이용하여 크롬을 실행시켜 미리 지정한 키워드를 넣고 검색, 페이지 마다 들어가 text를 크롤링하는 식으로 크롬 10여개의 페이지 크롤링하고 결과는 엑셀이나 메모장으로 정리한 후 다시 명사 순으로 많이 나온 순서 정리하여 파일 저장
